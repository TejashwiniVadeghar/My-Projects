## Image Caption Generator - Open Lab Project
(Feb 2020 – Jun 2020) 
Team: Tejashwini Vadeghar, Pranavi Marupudi, K Chandana Priya, Harshitha V, Shruthi Y, Vaishnavi Velidi

This project was done as part of Open Lab Project.

Description: Using Flickr_8K dataset, built the working model by using CNN (Convolutional Neural Networks) and LSTM (Long short term memory) units and developed using python. Data Pre-processing and Data Cleaning is done. Dataset is Trained and tested in the ratio of 75:25. Similar set of images will group together and a caption is generated which is relevant to grouped images.
